# dmc-wallets-tool
A tool to fast generate wallets for DMC

## How to use

### How to Install

clone the repo and install the dependencies

```bash
yarn install
yarn serve
```

* All Generate Logic In File  `src/wallet/index.js`

### App Demo Site
